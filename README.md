# 关于SH600516方大炭素的风险收益特征与行业基本面分析
本次分析主要是想通过近5年方大炭素的股票数据来展示该股票的历史表现并简要分析风险水平、行业内的盈利稳定性以及与市场的关联性。
首先通过导入Akshare的相关数据分析近5年方大炭素这只股票的历史每日表现与交易量的表现，发现改制股票基本也符合价格与交易量是属于正相关的，价格较高的时间其交易量也较高，但是该股票今年处于价格低位。
其次则计算了股票近5年的总收益率为-10.12%，年化收益率为-2.181%，这表明该支股票近5年收益率一直为负。此外，该支股票的最大回撤率71.39%，说明这只股票经历了比较大的跌幅。而该股票的夏普比率为-0.54则说明该投资者承担风险后获得的回报甚至不如持有无风险资产。
同时通过横向通过可视化来对比同行业其他7家企业的ROE来确定该企业在行业的具体位置，经过图标展示发现该支股票的ROE是低于其他企业，说明在石墨及碳素制品板块，该企业的盈利能力较弱。
最后则计算了了该支股票与沪深300指数的相关系数，其为0.43，说明该支股票与沪深指数有一定的相关性，但并不高。
总之，通过以上分析可以发现该支股票的各项指标比较不容乐观，**不建议持有这只股票**。未来，则可以考虑引入更多的财务指标，包括过去该支股票3-5年的PE及其历史分位数来更加客观而全面反映该支股票的情况。
